## Workday Scheduler
## There are 3 main functions
## Sethours will show the the 9am - 5pm hours on the left side of the calendar. Moments is used to ensure that the numbers are in the appropriate format
## Calcolors is used to make sure that all the past, present and future colors are accordingly defined. It compares current time with time on calendar entry and accordingly styles that entry . There in additional function calCals to add the actual past,present or future classes.
## save function to store the calendar entry to local storage
## This entire program is run in a loop. Every 5 mins, Workday Scheduler checks to see the current time and adjust the colors of the calendar entries. We use setInterval just in case the user leaves calendar open without refreshing for an extended period of time
